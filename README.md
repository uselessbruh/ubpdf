# UB PDF - All-in-One PDF Toolkit

A powerful desktop application for all your PDF manipulation needs. Built with Electron for a smooth cross-platform experience.

## üöÄ Features

### üìÑ PDF Operations

#### **Merge PDFs**
- Combine multiple PDF files into a single document
- Drag and drop interface for easy file management
- Reorder pages before merging
- Preview file details (name, size, page count)
- Remove unwanted files from the merge list

#### **Split PDF**
- Extract specific pages or page ranges from PDFs
- Split by single pages, page ranges, or custom selections
- Multiple split modes:
  - Extract specific pages (e.g., 1, 3, 5)
  - Extract page ranges (e.g., 1-5, 10-15)
  - Split into individual pages

#### **Organize PDF**
- Rearrange pages within a PDF
- Delete unwanted pages
- Rotate pages (90¬∞, 180¬∞, 270¬∞)
- Duplicate pages
- Visual page preview with thumbnails

#### **Add Watermark**
- Add text or image watermarks to PDFs
- Customize watermark properties:
  - Text content and font size
  - Opacity control (0-100%)
  - Position (center, top-left, top-right, bottom-left, bottom-right)
  - Rotation angle
- Support for PNG/JPG image watermarks

#### **Add Page Numbers**
- Automatically number PDF pages
- Customize numbering:
  - Starting number
  - Font size
  - Position (top-left, top-center, top-right, bottom-left, bottom-center, bottom-right)
  - Page range selection
- Format: "Page X of Y"

#### **Convert PDF to Images**
- Export PDF pages as high-quality images
- Supported formats: PNG, JPEG
- Configurable DPI (72-300)
- Options:
  - Convert all pages or specific pages
  - Batch export with automatic numbering

#### **Convert Images to PDF**
- Create PDFs from multiple images
- Supported formats: PNG, JPG, JPEG, BMP, GIF
- Drag and drop multiple images
- Automatic page sizing

#### **Protect PDF**
- Secure PDFs with password encryption
- Two password levels:
  - User password (open document)
  - Owner password (restrict editing)
- Set permissions:
  - Prevent printing
  - Prevent copying content
  - Prevent annotations
  - Prevent form filling

#### **Unlock PDF**
- Remove password protection from PDFs
- Support for user and owner passwords
- Quick and easy decryption

#### **Compress PDF**
- Reduce PDF file size
- Adjustable image quality (0-100%)
- Compression options:
  - Screen (72 DPI)
  - eBook (150 DPI)
  - Printer (300 DPI)
  - Prepress (300 DPI, high quality)
- Shows original vs compressed file size

#### **Edit Metadata**
- View and modify PDF properties
- Editable fields:
  - Title
  - Author
  - Subject
  - Keywords
  - Creator
  - Producer
  - Creation date
  - Modification date

### üîÑ Document Conversion

#### **PDF to Word**
- Convert PDF documents to editable DOCX files
- Preserves formatting, fonts, and layout
- Maintains document structure

#### **Word to PDF**
- Convert Word documents (DOCX) to PDF
- Preserves text formatting and styles
- Maintains paragraph structure

#### **PDF to Excel**
- Extract tables from PDF to Excel spreadsheets
- Automatic table detection
- Exports to XLSX format
- Preserves table structure and data

#### **Excel to PDF**
- Convert Excel spreadsheets (XLSX) to PDF
- Preserves formatting and table structure
- Includes grid lines and cell borders

#### **PDF to HTML**
- Convert PDF to web-ready HTML
- Preserves text formatting (bold, italic, headings)
- Extracts tables with proper styling
- Maintains font sizes and styles
- Responsive design with clean CSS

#### **HTML to PDF**
- Convert HTML files to PDF documents
- Supports tables with headers and borders
- Preserves headings (H1-H6)
- Maintains bold and italic formatting
- Professional page layout

#### **PDF to PowerPoint**
- Extract PDF content to PowerPoint slides
- Creates PPTX presentations
- One page per slide

#### **PowerPoint to PDF**
- Convert PowerPoint presentations (PPTX) to PDF
- Preserves slide layouts and content
- Maintains text formatting
